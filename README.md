# MarkerKit
Lightweight and easy to use wrapper for Auto Layout Constraints, inspired by: [https://github.com/ustwo/autolayout-helper-swift](https://github.com/ustwo/autolayout-helper-swift)

## Requirements

- iOS 9.0+
- Xcode 9.0+
- Swift 4.0+
